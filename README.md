# Jibot
- My personal discord bot I use for things I wanted to make

Current Features:
=====

## slash commands:
- ping: sends your ping
- dox: sends a random coordinate unrelated to the person
- user: sends what time they joined the server
- server: sends name and member count of server# Jibot
